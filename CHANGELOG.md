# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024

### Added
- Initial release of Maanjoo Farms Backend
- Product CRUD (admin) + public listing
- Room/Villa CRUD (admin) + public listing with search, filter, pagination
- Room availability check
- Shopping cart management
- Product ordering with Razorpay payments
- Villa booking with Razorpay payments
- Payment verification (client-side + webhook)
- Email notifications for orders, bookings, and contact forms
- Gallery management
- Pincode validation (India Post API)
- Blocked pincode management
- JWT authentication with admin/user roles
- File uploads (images) for products and gallery
- PDF invoice generation utility (unused)
- MongoDB TTL index for auto-cleaning pending bookings
